Project Description
This app was designed for language practice through multiple mediums, including an AI-enabled chat for dynamic learning and immediate feedback. The front end was developed using WinUI 3 and XAML, while the backend was built with C# and Python. The AI system utilizes the Google Gemini API through a Python script and Flask server, hosted on an external cloud platform. The app stores persistent data for users to track both overall and daily practice stats, and includes mistake-logging to support targeted improvement.

Project Description
This is a Java-based project created in 3 weeks for a presentation to the public. I worked as the lead programmer and designer, managing a team of 6 to create a turn-based fighting game with a variety rooms, enemies, and weapons. This project additionally utilizes libGDX, a game development framework, for rendering systems.
